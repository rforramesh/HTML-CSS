*****action="" :- here we specify our backend api

****While creating radio button or checkbox,we need to specify their resp. values using value="" and with same name="" attribute
****Imp :- By default,html form sends data as formData() to backend as formData can store file inputs We know, formData() obj is like hashmap i.e. key-value pair data structure(js in-built). Therefore,we must Always include name="" attribute inside input tag because name="" acts as key & value="" acts as value for formData() obj that will be sent to backend.
