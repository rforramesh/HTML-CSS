1. Sibling selectors : are used to target elements that are siblings i.e. they share the same parent.
There are 2 types of sibling selectors : adjacent sibling selector(+) and general sibling selector(~).

2. adjacent sibling selector(+) : targets an element that is immediately after current element in the DOM.

3. general sibling selector(~) : targets all siblings that appear after current element in the DOM.

Note : Sibling selectors can only select elements that appear after current element in the DOM.
