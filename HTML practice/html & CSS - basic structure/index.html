I am article tag used to write article, blog, post, etc.

We can include script inside head tag or at the bottom of body tag. if script is included inside head,then it blocks HTML rendering i.e. it first loads & executes the script and then the HTML will render. So we use keywords like async or defer with script tag inside head tag to achieve parallel loading of scripts & HTML rendering. [ Best practice : scripts that doesn't interact with current html page like angualrjs, jquery, bootstrap,other libraries,etc include it inside head tag with async keyword. scripts that interacts with current html page like myscript.js, include it at the bottom of the body tag. ]
