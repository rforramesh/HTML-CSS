Notes :-
Specificity means how much we are specific while target a element. Highest specificity will win & its style will be applied to the element.

Specificity value :
!important (10,000) > inline css (1000) > #id selector (100) > .class selector/[attribute selector]/:pseudo-classes (10) > element selector (1) > universal selector i.e. * (0)

// ================== CSS selectors ================================
* {} // universal selector
div {} // element i.e. tag selector
[id] {} OR input[type="submit"] {} // attribute selector
[data-color="primary"] {} //attribute selector-we can write & select our custom attribute
#box {} // id selector
.box {} // class selector
div > p {} OR .container > p{} OR .container > .box{} // child selector (direct-child)
div p {} OR .container p{} OR .container .box{} // descendent selector (any-child)
.box * {} // to apply to all child elements
.box::first-line {} OR .box::first-letter {}
.box:first-child {}
.box:last-child {}
.box:nth-child(1) {}
.box:nth-of-type(even) {}
p:nth-last-child(2) {} // niche se second child
.box::before {content: " ";} OR .box::after {content: " ";} //pseudo elements
.box::selection {}
